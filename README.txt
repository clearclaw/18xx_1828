1828: An 18xx game set in the Eastern States of America which features
16 public companies, unusual private companies, mergers, a novel stock
market, no certificate limit and several other curiousities.

A produced copy of the files ready for printing your own copy can be
found at: https://kanga.nu/~claw/1828/
